# Sahab Saeed  Personal Portfolio Website

A clean, responsive, and modern personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks, no libraries — just clean code.

---

## About This Project

This is my personal portfolio website that I built to showcase my skills, projects, and experience as a Software Engineering student and Front-End Developer intern at DevelopersHub Corporation. The goal was to create a professional, fully responsive website using only core web technologies — HTML, CSS, and JavaScript.

---

## Live Demo

Live Demo: https://sahab-saeedportfolio.netlify.app/

---

## Technologies Used

- **HTML5** — for the structure and content of the website
- **CSS3** — for all styling including Flexbox, Grid, animations, and responsive design
- **JavaScript (Vanilla)** — for all interactivity including typewriter effect, scroll animations, skill bar animations, hamburger menu, and form validation
- **Google Fonts (Poppins)** — for typography
- **Font Awesome** — for icons

---

## Features

- Fully responsive design that works on mobile, tablet, and desktop
- Typewriter animation that cycles through different roles
- Smooth scroll navigation between sections
- Animated skill bars that trigger when the section comes into view
- Scroll-triggered fade-up animations on cards and sections
- Hamburger menu for mobile navigation
- Contact form with basic input validation
- CV download button linked to a PDF file
- Custom navy blue and white color theme with gold accents

---

## Sections

**Home**
The landing section includes a profile photo, my name, a typewriter animation showing my roles, a short introduction paragraph, and two call-to-action buttons — Hire Me and Download CV.

**About**
This section contains my profile photo, a short bio, and a details grid with information like my name, email, university, CGPA, degree, location, internship status, and freelance availability.

**Skills**
Displays animated skill bars for the technologies I work with — HTML, CSS, C++, Java, MySQL, and Responsive Design. Each bar animates when scrolled into view.

**Services**
Three service cards describing what I offer — Web Development, UI/UX Design, and Software Development — each with a relevant icon.

**Contact**
A contact form with fields for name, email, subject, and message, along with a submit button that shows a success message on valid submission.

## How to Run Locally

1. Clone this repository
   ```
   git clone https://github.com/534670/sahab-saeed-portfolio.git
   ```

2. Navigate to the project folder
   ```
   cd sahab-saeed-portfolio/port_folio
   ```

3. Open `index.html` in any browser, or use the Live Server extension in VS Code for auto-reload on changes.
